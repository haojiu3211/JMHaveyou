基础免费服务、VIP 会员服务以对应专项协议为准，活动币付费增值规则优先适用本协议，条款冲突时以本协议为准。

3.2 活动币可解锁付费特权（仅消耗活动币使用）

持有活动币余额时，可消耗对应数量活动币使用以下增值功能，免费用户、无余额 VIP 无法解锁：

无限畅聊活动细节特权：向任意活动发布者发起不限次数深度咨询、置顶消息、多轮私信沟通，单次畅聊权限消耗固定活动币；

配套增值场景（平台后续迭代拓展，提前公示消耗标准）：活动置顶曝光、批量访客隐身查看、批量用户定向邀约等。

活动币不能用于购买连续包月 VIP 会员、不能抵扣会员订阅费用；会员与活动币为两套独立付费体系；

四、充值、消耗、余额查询操作规则

充值流程：APP 内「我的 - 活动币余额 - 充值」→ 选择档位 → 验证 Apple ID 密码 / Face ID/Touch ID 完成支付 → 支付成功实时发放活动币至当前账号；支付失败不扣款、不发放活动币；

消耗流程：使用一呼百应、无限畅聊等付费功能时，页面提前展示本次消耗活动币数量，用户手动确认后扣减余额；余额不足时无法执行对应功能；

余额与账单查询：用户可在「我的 - 活动币 - 账单明细」查看全部充值记录、活动币消耗记录、剩余余额，账单永久留存，用户可随时截图留存；

补发规则：出现支付扣款成功但活动币未到账的情况，用户可提供苹果订单收据联系客服，平台核验无误后 1-3 个工作日补发对应活动币，不提供现金退款。

退款审核主体为苹果公司，平台无权限直接操作充值退款、撤销 Apple ID 订单，所有退款申请仅能通过苹果官方渠道提交：App Store 右上角头像→账户→购买记录→对应充值订单→报告问题。

不予退款标准场景（苹果统一规则）

（1）活动币已部分 / 全部消耗，已使用一呼百应、无限畅聊等付费功能；

（2）用户自愿充值、功能正常无故障，仅主观不愿继续使用；

（3）用户自身忘记未成年人管控、主动给他人代充后反悔；

（4）账号因发布违规活动、骚扰他人、私下交易活动币被平台处罚封禁；

（5）超过充值 90 天的历史订单，苹果关闭退款申诉通道。

可提交退款申诉场景（需在扣款 14-90 天内提交完整订单凭证）

（1）重复扣款、系统故障扣款后活动币始终未到账；

（2）未成年人未经监护人许可私自大额充值；

（3）APP 核心付费功能长期故障，完全无法使用活动币对应特权。

退款后果：若苹果审核通过退款，平台将同步扣除对应充值到账的活动币余额；余额不足抵扣的，平台有权临时限制账号付费功能直至差额结清。
